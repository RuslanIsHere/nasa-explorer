# NASA Explorer

**NASA Explorer** is a small educational project built with **Next.js**, **TypeScript**, and **Material UI**, designed to explore and display data from various public **NASA APIs**.  
The app demonstrates how to fetch and render external data, handle async operations, and structure a simple modern frontend with server-side rendering.

## Features

- Fetches and displays the **Astronomy Picture of the Day (APOD)** from NASAâ€™s open API.
- Responsive UI styled with **Material UI** components.
- Fully typed with **TypeScript**.
- Simple project structure for practicing Next.js basics.

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Material UI](https://mui.com/)
- [NASA APIs](https://api.nasa.gov/)
